# 10. Introduction to JavaScript

## Primitive Datatypes

* Numbers \(including whole numbers, fractional numbers, etc.\)
  * `4`
  * `9.3`
  * `-10`
* Strings
  * `"Hello World"`
  * `"43"`
  * The backslash character `\` is the **escape character** in JavaScript.
* Booleans
  * `true`
  * `false`
* null
* undefined
