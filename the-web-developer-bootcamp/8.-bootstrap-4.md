# 8. Bootstrap 4

* Font size unit changed to relative \(to root size: 16px by default\)
* Many font colors offered by the attribute `class`
  * `primary`
  * `secondary`
  * `success`
  * `danger`
  * `info`
  * `...`
* Border shape offered by the attribute `class`
* **Responsive breakpoints**
  * `@media`
